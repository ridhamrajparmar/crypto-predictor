# AI-Powered Cryptocurrency Predictor

A web-based dashboard that uses **LSTM Neural Networks** to predict the long-term prices of the top 30 cryptocurrencies.

## Features
- **AI Forecasting:** 7-day price predictions using deep learning.
- **Interactive Dashboard:** Clean, terminal-style UI built with Flask and Chart.js.
- **Dataset:** Analyzes historical data from the top 30 coins.

## Tech Stack
- **Backend:** Python (Flask)
- **Frontend:** HTML5, CSS3, JavaScript (Chart.js)
- **AI/ML:** TensorFlow/Keras (LSTM), Pandas, NumPy
- **API:** CoinGecko (for metadata)

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `python app.py`
3. Open `http://127.0.0.1:5000` in your browser.
